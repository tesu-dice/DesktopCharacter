    アプリケーションの利用方法と設定等に関するガイド

    概要

    はじめに

    このアプリケーションは、デスクトップ上にAIキャラクターを召喚し、パソコンでの活動を少し賑やかにするためのツールです。
    各種設定（最低でもAIの項目の設定が必要です。）を行うことでAIとの対話やあなたのPC利用状況に応じたコミュニケーションを楽しむことができます。

    動作環境
    OS: Windows 10 / 11
    ※セキュリティソフトに身元不明のアプリとして誤検知される可能性があります。

    主な機能
    • AIキャラクターとの自由な会話
    • 音声合成による読み上げ機能
    • 好みの立ち絵（透過画像）の表示と、表情差分への対応
    • AIキャラクターへのキャラ付けや規則の自由な変更
    • ユーザアクティビティの記録機能

    プライバシーポリシー

    アプリケーション内におけるユーザの個人情報の扱いについて以下の通りとします。
    利用する情報
    アプリケーションは以下の情報を取得・収集・送信します。 これらの項目は設定画面によって許可、制限を行うことができます。
      a. ユーザがアプリケーション内で入力したテキスト
      b. ユーザがアプリケーションを利用している時刻情報
      c. アクティブウィンドウのタイトル名
      d. 再生中のメディア情報(曲名、アーティスト名)
      e. 上記b, c, dの項目を使ったユーザ情報の収集
    情報の利用目的
    取得した情報はアプリケーション内でのユーザ体験を目的としてのみ利用され、 開発者がこれらの情報を収集、保存、閲覧することはありません。

    情報の第三者提供
    会話動作、音声読み上げの際に、取得した情報を連携外部サービスへと送信します。 各サービスでの個人情報の扱いはは各サービスのプライバシーポリシーをご確認ください。

    AIサービスの設定

    アプリケーションを通した会話の際に利用するAIサービスの設定案内です。
    会話の動作にはどちらかのサービスに接続する必要があります。 設定ウィンドウのAIサービスの選択項目で、GeminiかOllamaのどちらかを選択して利用してください。

    Google Gemini

    Gooleの提供しているAIサービスGeminiを使った方法です。Googleアカウントが必要です。
    今回の手順では無料枠での利用となり、利用上限の設定と送信データの学習利用が行われます。

    1. Google AI StudioでAPIキーを取得する

    1. Webブラウザで Google AI Studio にアクセスします（Googleアカウントが必要です）。
    2. 画面左側のメニューから「Get API key」をクリックします。
    3. 「Create API key in new project」ボタンをクリックして、新しいAPIキーを作成します。
    4. APIキーが生成されたら、コピーボタンをクリックしてクリップボードにコピーします。
      例: AIzaSy...（このような長い文字列です）
    注意事項
    このAPIキーはパスワードと同じように扱い、他人に知られないように大切に保管してください。
    流出や不正利用が疑われる場合はAPIキーを停止、削除する等の対応が必要になります。

    2. アプリケーション側の設定を行う

    1. キャラクターを右クリックし「設定」を選択して、設定ウィンドウを開きます。
    2. 設定ウィンドウ内の「AIサービスの選択」の項目をGeminiに設定します。
    3. 設定ウィンドウ内の「Gemini」の項目を探します。
    4. APIキーの項目にコピーしたAPIキーを貼り付け、設定を保存、アプリを再起動します。
    5. 再度設定ウィンドウを開いてGeminiの項目、モデルの隣にある更新ボタンを押した後、会話モデルを選択します。
      6. 2026年1月時点ではgemini-2.5-flash-lite等での動作確認ができています。
      Googleによって提供モデルの変更が行われる場合があるためその際は都度モデルの変更等の対応が必要になります。
    6. アプリを再起動し、起動時のメッセージでAIサービスとの接続項目が成功になっていれば完了です。

    Ollama

    AIモデルをローカル環境で動作せる無料ツール、Ollamaと連携する方法です。 無料かつローカル環境で動作するため情報流出等を気にしすぎなくて済みます。

    1. Ollamaのインストール

    1. Ollama公式サイト からインストーラーをダウンロードし、インストールします。(手順省略)
    2. インストール後、Ollamaを起動してください。タスクトレイにアイコンが表示されます。
      3. Ollamaが起動している(タスクトレイに表示されている)状態で連携が行えます。利用の際はOllamaはアプリと別で別途起動してください。

    2. モデルのセットアップ

    1. コマンドプロンプト・ターミナルを開きます。
    2. ollama pull モデル名とコマンドを実行してモデルをダウンロードします。おすすめはgemma3です。
      例：ollama run gemma3
      利用可能なモデルは Ollamaモデルライブラリ で確認可能です。 PU性能や日本語対応などある程度知識や調べものが必要になると思います。
    3. ollama listコマンドでモデルが表示されていればモデルセットアップは完了です。

    3. アプリ側での設定

    1. キャラクターを右クリックし「設定」を選択して、設定ウィンドウを開きます。
    2. 設定ウィンドウ内の「AIサービスの選択」の項目をOllamaに設定します。
    3. 設定ウィンドウ内の「Ollama」の項目を探します。
    4. OllamaAPIのURL の項目に http://localhost:11434 が入力されているか確認します。
      5. Ollam側で設定されている外部連携用のURLです。デフォルトでは上記の値ですが、変更している場合は対応させてください。
    5. モデルの項目で、更新ボタンを押した後モデルを選択、設定を保存します。
    6. アプリを再起動し、起動時のメッセージでAIサービスとの接続項目が成功になっていれ完了です。

    音声読み上げの設定

    アプリケーション内の設定ウィンドウ、合成音声の項目で設定することができます。会話の際にセリフを合成音声で読み上げます。

    VOICEVOX

    無料で使える音声読み上げソフトウェアのVOICEVOXと連携する方法です。 お使いのパソコンにVOICEVOXをインストールし、アプリ側から起動することで動作させます。

    1. VOICEVOXのインストール

    1. VOICEVOX公式サイト からインストーラをダウンロードし、インストールします。(手順省略)
    2. VOICEVOXを起動します。VOICEVOXが起動していれば連携が可能ですが、次の手順での設定によってはアプリ起動時に自動で起動するようにもできます。

    2. アプリケーションの設定

    1. キャラクターを右クリックし「設定」を選択して、設定ウィンドウを開きます。
    2. 設定ウィンドウ内の「合成音声エンジンの選択」の項目をVOICEVOXに設定します。
    3. 設定ウィンドウ内の「VOICEVOX設定」の項目を探します。
    4. 音声モデルの項目で、更新ボタンを押した後モデルを選択、設定を保存します。自動起動を設定しない場合は最後の手順まで飛んでかまいません。
    5. VOICEVOXの起動パス の項目でVOICEVOXEngine(VOICEVOXのサーバ)の起動ファイル(run.exe)がを設定します。
      6. 標準設定でインストールをした場合、次のどちらかのパスにあるかと思います。
      1. C:\Program Files\VOICEVOX\vv-engine\run.exe
      2. C:\Users\[あなたのユーザー名]\AppData\Local\Programs\VOICEVOX\vv-engine\run.exe
    6. アプリを再起動し、会話の際に音声読み上げが行われていれば完了です。自動起動設定をしている場合は起動メッセージで成否が確認できます。

    Windows標準ナレーター

    Windowsの音声読み上げサービスを利用する方法です。

    1. アプリケーションの設定

    1. キャラクターを右クリックし「設定」を選択して、設定ウィンドウを開きます。
    2. 設定ウィンドウ内の「合成音声エンジンの選択」の項目をWindowsNarratorに設定します。
    3. 設定ウィンドウ内の「WidnowsNarrator設定」の項目を探します。
    4. 音声モデルの項目で、更新ボタンを押した後モデルを選択、設定を保存します。
    5. アプリを再起動し、会話の際に音声読み上げが行われていれば完了です。

    そのほかの機能

    キャラクター画像の変更

    設定ウィンドウのキャラクター参照フォルダの項目から、キャラクター画像は入っているフォルダを選択することでキャラクター画像の変更を行うことができます。

    キャラクター画像の追加

    1. アプリケーションが入っているフォルダの「立ち絵」フォルダ内に、立ち絵セット名のフォルダを作成します。
    2. 作成したフォルダ内に表示したい立ち絵ファイル(.png)を入れます。
      画像ファイル名は、キャラクターの表情を表すものにしてください。(例：「平穏.png」「笑顔.png」「怒り.png」)

    キャラクター設定のテキスト編集

    アプリケーションが入っているフォルダ内の「Character_setting.txt」を編集することによって会話するキャラクターの設定を決めることができます。 このファイルの内容を参考に応答文章が作成されます。

    ユーザアクティビティの記録と活用機能

    5分ごとにアプリケーションで許可された項目を記録、1時間間隔で要約するなどしてアプリ内での会話で利用できるようにします。
    アプリケーションが入っているフォルダ内の「user_logs」フォルダに日ごとにファイルが作成されます。
    AIサービスの使用量が大きくなるため使用量・料金の増加に気を付けてください。
    また、ユーザの活動時間等の個人情報を多大に含むのでOllamaでの動作を推奨します。

    データの引継ぎ方法

    旧版から新版へデータを引き継ぐための手順です。 main.exe, Readmeのドキュメント以外のファイルを新版のフォルダへコピー、移動させます。
    各ファイルは場合によっては引き継ぎが行えない場合があります。ご了承ください。
