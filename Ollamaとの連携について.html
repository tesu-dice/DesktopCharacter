Ollama API との連携方法

このアプリケーションは、ローカル環境で大規模言語モデル（LLM）を実行できる「Ollama」と連携して、AI会話機能を利用することができます。 このドキュメントでは、Ollamaを導入し、アプリケーションと連携させるための手順を説明します。

ステップ1: Ollamaをインストールする

  1. まず、お使いのPCにOllamaがインストールされていない場合は、公式サイトからダウンロードしてインストールします。
    Ollama公式サイト
  2. お使いのOS（Windows, macOS, Linux）に合ったインストーラーをダウンロードし、画面の指示に従ってインストールを完了してください。

ステップ2: Ollamaで利用するモデルをセットアップする

  1. Ollamaをインストールしたら、次に会話に使用したいAIモデルをダウンロードします。コマンドプロンプト（またはターミナル）を開いて、以下のコマンドを実行します。
  2. 例えば、Meta社の「Llama 2」モデルをダウンロードするには、次のコマンドを入力します。
    ollama run llama2
  3. コマンドを実行すると、モデルのダウンロードが開始されます。完了すると、コマンドラインでそのモデルと直接対話できるようになります。
    補足: 他にも様々なモデルが利用可能です。利用可能なモデルのリストはOllamaのモデルライブラリで確認できます。

ステップ3: アプリケーションにOllamaの設定を行う

  1. デスクトップキャラクターを右クリックして表示されるメニューから「設定」を選択し、設定ウィンドウを開きます。
  2. 設定ウィンドウの上部にある「AI」の選択肢から「Ollama」を選びます。
  3. 「Ollama Settings」セクションで、以下の項目を設定します。
    • Ollama_base_url: Ollamaが動作しているURLを入力します。通常、Ollamaを自分のPCで動かしている場合は、デフォルトの http://localhost:11434 のままで問題ありません。
    • Ollama_model: ステップ2でセットアップしたモデル名（例: llama2）を入力します。
  4. 入力が完了したら、設定ウィンドウの下部にある「保存して閉じる」ボタンをクリックします。

設定完了

以上で設定は完了です。ローカル環境で動作するAIとの会話機能が有効になります。
