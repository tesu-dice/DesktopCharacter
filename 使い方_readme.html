はじめに

このアプリケーションは、あなたのデスクトップに常駐するキャラクターと会話を楽しむためのものです。AIとの対話や、あなたのPC利用状況に応じたコミュニケーションを楽しむことができます。

動作環境

  • OS: Windows 10 / 11
  • AI対話API: Google Gemini API (APIキーが別途必要です)
  • 音声合成エンジン: VOICEVOX または Windows標準ナレーター (VOICEVOXを利用する場合は別途インストールと起動が必要です)

プライバシーポリシー (Privacy Policy)

本アプリケーションは、キャラクターとの自然な対話を実現するために、以下の情報を取得・利用します。

1. 取得する情報

  • ユーザーが入力したテキスト: AIとの対話のために利用します。
  • アクティブウィンドウのタイトル名: キャラクターがユーザーの作業状況に応じた会話をするために利用します。 (設定でON/OFF可能)
  • 再生中のメディア情報（曲名、アーティスト名）: キャラクターが再生中の音楽について会話するために利用します。 (設定でON/OFF可能)

2. 情報の利用目的

取得した情報は、キャラクターとの対話内容をAIが生成する目的でのみ利用されます。

3. 情報の第三者提供

キャラクターとの対話内容を生成するため、取得した情報を暗号化された通信（HTTPS）を用いて以下の外部APIサービスに送信します。

  • Google Gemini API: ユーザーとの対話内容を生成するために、入力テキストやPC利用状況の情報を送信します。

開発者がこれらの情報を収集、保存、閲覧することはありません。外部APIサービスのプライバシーポリシーについては、各サービスの公式サイトをご確認ください。

インストールと初期設定

  1. アプリケーションをダウンロードし、任意のフォルダに展開します。
  2. Gemini APIキーの取得: Google AI Studio にアクセスし、APIキーを生成・コピーします。
  3. VOICEVOXの準備 (利用する場合): VOICEVOX公式サイトからVOICEVOXをダウンロードし、インストール・起動してください。
  4. アプリケーションの起動と設定:
    • アプリケーションを起動し、設定画面を開きます。
    • 「geminiAPIkey」の項目に、コピーしたAPIキーを貼り付けます。
    • 設定を保存します。APIキーが設定されていない場合、対話機能は利用できません。

キャラクター設定

キャラクター画像の変更

アプリケーションのインストールフォルダ内の「立ち絵」フォルダに、キャラクターの画像ファイル（.png）を配置します。画像ファイル名は、キャラクターの表情を表すものにしてください。（例：「通常.png」「笑い.png」「怒り.png」）

キャラクター設定テキストの編集

アプリケーションのインストールフォルダ内の「Character_setting.txt」ファイルをテキストエディタで開き、キャラクターの性格や口調などを記述します。この内容は、AIがキャラクターとして返答を生成する際の参考にされます。

# キャラクター設定の例
名前：てすこ
性格：明るく元気で、少しおっちょこちょい。ユーザーのPC作業を応援するのが好き。
口調：～だよ！、～だね！、～なんだ♪
一人称：わたし
好きなもの：新しいソフトウェア、面白い動画
嫌いなもの：フリーズ、エラーメッセージ

トラブルシューティング

  • キャラクターが話さない:
    • Gemini APIキーが正しく設定されているか確認してください。
    • インターネット接続が正常であるか確認してください。
    • VOICEVOXを利用している場合、VOICEVOXエンジンが起動しているか確認してください。
  • 設定が反映されない:
    • 設定を保存した後、アプリケーションを再起動してみてください。
    • 設定ファイル（config.json）が破損していないか確認してください。
