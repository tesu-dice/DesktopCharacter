Gemini APIキーの設定方法

このアプリケーションのAI会話機能を利用するには、Googleが提供する「Gemini API」のキーが必要です。 このドキュメントでは、APIキーを取得し、アプリケーションに設定するまでの手順を説明します。

ステップ1: Google AI StudioにアクセスしてAPIキーを取得する

  1. まず、WebブラウザでGoogle AI Studioにアクセスします。（Googleアカウントでのログインが必要です。）
  2. 画面左側のメニューから「Get API key」をクリックします。
  3. 「Create API key in new project」ボタンをクリックして、新しいAPIキーを作成します。
  4. APIキー（長い文字列）が生成されたら、コピーボタンをクリックしてクリップボードにコピーします。
    AIzaSy...（このような長い文字列です）
    重要: このAPIキーは、他人に知られないように大切に保管してください。パスワードと同じように扱う必要があります。

ステップ2: アプリケーションにAPIキーを設定する

  1. デスクトップキャラクターを右クリックして表示されるメニューから「設定」を選択し、設定ウィンドウを開きます。
  2. 設定ウィンドウ内の「ApplicationSettings」セクションを探します。
  3. 「geminiAPIkey:」という項目を見つけ、その右側にある入力欄に、先ほどコピーしたAPIキーを貼り付けます。
  4. 入力欄にキーを貼り付けたら、設定ウィンドウの下部にある「保存して閉じる」ボタンをクリックします。

設定完了

以上で設定は完了です。AIとの会話機能が有効になります。
